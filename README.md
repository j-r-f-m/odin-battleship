# Schiffe-Versenken-App

Mit der Schiffe-Versenken-App kann man eine Runde Schiffe versenken gegen eine einfach KI spielen. Nach dem Platzieren der Schiffe mit der Maus kann das Spiel gestartet werden. Jedes Schiff kann gedreht werden. Klicke auf ein Feld um es zu beschie√üen. Wird das Feld rot, wurde ein feindliches Schiff getroffen. Wird das Feld blau, wurde nichts getroffen. Das Spiel endet, wenn der alle Schiffe des Spielers oder der KI getroffen wurden.
